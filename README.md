# Webtrees Svajana Theme

A custom Webtrees theme module designed to integrate seamlessly with the WordPress Kadence Child Theme.

## Features
- **Split-Stack Header**: Merges Webtrees navigation with WordPress branding.
- **Mobile Drawer**: Full off-canvas mobile menu matching Kadence behavior.
- **Shared Database**: Pulls configuration (Colors, Fonts) directly from WordPress `wp_options`.
- **Footer Sync**: Replicates the Kadence footer layout.
- **Customizable**: Includes `custom.css` for safe overrides.

## Installation

1. Create a directory in your Webtrees installation:
   `modules_v4/webtrees-svajana`

2. Copy the files generated by the Architect tool into this folder following the structure:
   ```
   webtrees-svajana/
   ├── module.php
   ├── WebtreesSvajana.php
   ├── README.md
   └── resources/
       ├── css/
       │   ├── base.css
       │   ├── custom.css
       │   └── webtrees-menus.css
       ├── js/
       │   └── theme.js
       ├── lang/
       │   └── (Optional .po files)
       └── views/
           └── layouts/
               ├── default.phtml
               ├── _menu.phtml
               └── _footer.phtml
   ```

3. Login to Webtrees Administration -> Modules -> Themes.
4. Enable "Webtrees Svajana".
5. (Optional) Set as default theme in Site Configuration.

## Customization
- To override styles without editing core theme files, add your CSS to `resources/css/custom.css`.
- To translate, add .po files to `resources/lang/`.
